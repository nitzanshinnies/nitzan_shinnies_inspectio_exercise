# v1_obsolete

This folder contains legacy v1 artifacts that are no longer part of the active implementation direction.

- `plans/` contains superseded architecture/performance notes kept for historical reference.
- `meta/` contains temporary working artifacts that are not part of the product.

Archived **v2** planning documents live in `v2_obsolete/plans/` (blueprint, phases, SQS FIFO throughput plan). Active **v3** design and `plans/openapi.yaml` are the integration surface going forward; see root **`README.md`**.
